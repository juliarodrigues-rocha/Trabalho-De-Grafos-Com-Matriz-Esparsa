# ✈️ Trabalho de Grafos com Matriz Esparsa
 
Este repositório reúne o **Trabalho de Grafos** desenvolvido por um grupo de **3 estudantes** do curso de **Sistemas de Informação** durante o terceiro semestre.  
O objetivo é aplicar, de forma prática, os conceitos de **grafos**, **matriz esparsa** e **árvore** estudados em sala de aula, desenvolvendo um sistema de organização da malha aérea brasileira para a ANAC.

## ▶️ Como rodar o projeto
 
É necessário ter o **GCC** instalado. 

Verifique com `gcc --version`.
 
Abra o terminal e siga os passos:
 
1 - `git clone https://github.com/juliarodrigues-rocha/Trabalho-De-Grafos-Com-Matriz-Esparsa.git`

2 - `cd Trabalho-De-Grafos-Com-Matriz-Esparsa`

3 - `gcc -o programa main.c aeroporto.c matriz_esparsa.c grafo.c`

4 - `./programa`

 
> No Windows: `programa.exe`
 
Prontinho, o sistema vai estar rodando!
